class Ex01 {}
- 클래스 이름은 저장된 이름과 무조건 같아야 한다.
- 클래스 이름의 첫글자는 무조건 대문자이다. Ex01, String, System 등등
- 클래스는 자바에서 저장할 수 있는 최소 단위이다.
- 클래스는 메서드와 필드로 구성되어있다.
- 필드: 상태값 = 데이터, 아는 것 ex) 눈, 코, 입, 귀
- 메서드: 동작 = 하는 것 ex) 먹고 자고 싸고
- 필드는 변수(변할 수 있는 상태값)와 상수(변하지 않는 상태값)로 구성

public static void main(String[] args) {}
- ()는 메서드이다.
- () 앞에는 메서드의 이름이다. 즉 main이 이름이다.
- 메인메서드는 JVM이 호출해서 사용하는 메서드이다. 
프로그램을 시작시키는 시작점이다.(Start Point)
- 모든 프로그램에는 main 메서드가 하나만 존재한다.
(0개 또는 여러개 이상 있으면 에러발생)

System.out.println("자바의 꿈");
System => class
out => field
println => method
- API 따르면 System.out => 모니터를 말함(표준 출력장치)
- System.in = > 키보드를 말함(표준 입력장치)
- println("내용") => 내용 출력
